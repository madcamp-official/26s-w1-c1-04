# 기능 명세서 (Feature Specification)

## 1. 맵 구조

### 1.1 개요

맵은 **Graph** 구조로 구성된다.

- Vertex : Room
- Edge : Room 간 이동 가능한 경로

플레이어는 Edge를 통해 Room 사이를 이동한다.

---

### 1.2 방(Room)

#### 정의

방(Room)은 맵의 Vertex에 해당하는 퍼즐 단위이며, 플레이어가 하나의 공간으로 인식하는 영역이다.

#### 방 종류

- 복도 (Corridor)
- 탕비실 (Pantry)
- 몰입 강의실 (Immersion Classroom)
- 열린 공간 (Open Space)
- 관리실 (Office)

#### 맵 연결 구조

- 복도는 모든 방과 연결된다.
- 일반 방은 복도와만 연결된다.
- 일반 방끼리는 직접 연결되지 않는다.

---

### 1.3 방 구성 요소

#### 플레이어에게 표시되는 요소

- 상호작용 가능한 도구(Object)
- 방을 나갈 수 있는 최종 문(Door)

#### 플레이어에게 표시되지 않는 데이터

- Room ID
- 연결된 Edge ID

---

### 1.4 Edge

Edge는 두 Room 사이의 이동 가능 여부를 나타낸다.

Edge는 다음 두 가지 상태를 가진다.

- Locked
- Unlocked

게임 시작 시 모든 일반 방으로 향하는 Edge는 Locked 상태이다.

지정된 퍼즐을 해결하면 대응되는 Edge가 Unlocked 상태로 변경된다.

Unlocked된 Edge는 게임이 종료될 때까지 다시 Locked 상태로 변경되지 않는다.

Edge가 Unlocked되면

- 해당 방향으로 이동이 가능해진다.
- 이동 가능한 방향에 화살표 이미지를 표시하여 플레이어에게 안내한다.

---

# 2. 이동

## 2.1 이동 방식

거리뷰(Street View) 방식의 이동을 사용한다.

플레이어는 자유롭게 이동하지 않으며,
이동 가능한 방향을 선택하여 해당 위치로 즉시 이동한다.

---

## 2.2 이동 방법

플레이어는 다음 두 가지 방법으로 이동할 수 있다.

- 화면에 표시된 이동 화살표 클릭
- 방향키 입력

선택한 방향으로 즉시 화면이 전환된다.

---

## 2.3 방 입장

플레이어는 방에 입장하면 항상 미리 지정된 위치와 시점에서 시작한다.

같은 방을 다시 방문하더라도 동일한 위치와 시점에서 시작한다.

---

# 3. 도구(Object)

## 3.1 정의

도구(Object)는 방 안에서 플레이어가 클릭하여 상호작용할 수 있는 객체이다.

모든 도구는 고유한 Object ID를 가진다.

---

## 3.2 도구 종류

### 힌트 도구

상호작용 시

- 해당 Object ID에 연결된 설명 또는 힌트를 표시한다.

힌트는 횟수 제한 없이 다시 확인할 수 있다.

---

### 게임 도구

상호작용 시

- 해당 Object ID에 연결된 미니게임을 실행한다.

---

## 3.3 상호작용 화면

도구와 상호작용하면 별도의 화면이 표시된다.

해당 화면에는

- 돌아가기 버튼
- 닫기(X) 버튼

이 존재한다.

버튼을 선택하면 현재 방 화면으로 복귀한다.

---

# 4. 퍼즐

## 4.1 방 퍼즐

복도에서 아직 Edge가 Locked 상태인 방으로 이동을 시도하면

- 퍼즐 이미지를 표시한다.
- 퍼즐 이미지 아래에 도어락 오브젝트를 표시한다.

플레이어는 퍼즐을 해결하여 해당 방으로 이동할 수 있다.

---

## 4.2 퍼즐 결과

### 성공

- 해당 퍼즐을 완료 상태로 변경한다.
- 연결된 Edge를 Unlocked 상태로 변경한다.
- 이동 가능한 방향에 화살표를 표시한다.

### 실패

- 퍼즐 화면을 유지한다.
- Edge 상태는 변경되지 않는다.

### 종료

플레이어는 언제든 퍼즐 화면을 닫고 복도로 돌아갈 수 있다.

---

# 5. 미니게임

미니게임은 특정 게임 도구와 상호작용할 때 실행된다.

예시

- 모니터
- (추후 추가 예정)

각 게임 도구는 하나의 미니게임과 연결된다.

---

## 5.1 미니게임 종료

미니게임 종료 시

- 미니게임 화면을 닫는다.
- 플레이어는 원래 방 화면으로 복귀한다.

미니게임의 성공 여부는 해당 미니게임의 구현에 따라 결정된다.

---

# 6. 관리자 모드

개발 및 테스트를 위한 기능이다.

현재 위치한 방에서 관리자 비밀번호를 입력하면 현재 방의 퍼즐을 강제로 해결한다.

### 관리자 비밀번호

```
1234
```

### 실행 결과

- 현재 방의 퍼즐을 완료 상태로 변경한다.
- 연결된 Edge를 Unlocked 상태로 변경한다.
- 실제 퍼즐은 수행하지 않는다.
