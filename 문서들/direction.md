# Direction Rules

## Horror

공포는 존재가 아니라 증거에서 나온다.

금지 - 괴물 - 귀신 등장 - 점프 스케어 남발 - 이유 없는 환각

허용 - 시간의 반복 - 기억의 모순 - 증거의 재해석 - 플레이어 자신의 흔적

------------------------------------------------------------------------

## Audio

-   평소에는 매우 조용하다.
-   특정 증거를 획득하면 이전에 들었던 음성이 새로운 의미로 들린다.
-   엔딩에서 모든 음성이 플레이어 자신의 목소리였음을 암시.

------------------------------------------------------------------------

## Lighting

초반은 현실적.

후반으로 갈수록 같은 공간인데도 미묘하게 조명이 달라진다.

루프가 겹치는 장소에서는 형광등 깜빡임보다 시간이 어긋난 듯한 연출을
우선한다.

------------------------------------------------------------------------

## UI

Notion GitHub VSCode Slack

실제 개발 환경과 최대한 동일하게 제작.

UI는 공포 연출이 아니라 증거 전달 수단이다.

------------------------------------------------------------------------

## Puzzle Rules

퍼즐은 열쇠를 주지 않는다.

퍼즐의 보상은 새로운 '사실'이다.

새로운 사실이 다른 방의 퍼즐 풀이가 된다.

------------------------------------------------------------------------

## Ending Rules

직접 설명 금지

절대 사용하지 않을 표현

-   자살
-   투신
-   죽으려 했다

플레이어가 모든 증거를 연결하여 스스로 결론을 내리도록 설계한다.
