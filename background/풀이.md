## 중급

### 중_5 — 방위(나침반) 암호 → **NEST** ✅

**문제:** 글자와 화살표가 섞인 격자.
```
T  H  →
A  ↑  ↓  ←  →  R
I  ↓
↑  →  ↓  T
```

**핵심 아하:** 화살표는 **나침반의 방위**이고, 방위의 영어 첫 글자로 바꾸면 된다.
- ↑ = North = **N**
- → = East = **E**
- ↓ = South = **S**
- ← = West = **W**

**풀이:** 화살표를 치환해 그대로 읽는다.
```
T H (E)              → THE
A (N)(S)(W)(E) R     → ANSWER
I (S)                → IS
(N)(E)(S) T          → NEST
```
문장은 **"THE ANSWER IS NEST"**. 문장 자체가 답을 말해 준다.

**정답: `NEST`**

### 중_6 — ON/OFF 스위치 삽입 → **OFFICE** ✅

**문제:** 토글 스위치 세 개.
| 스위치 | 상태 |
|---|---|
| 「S」 AND 「G」 | **ON** |
| 「C」 AND 「DOUBLE E」 | **OFF** |
| 「 」 AND 「ICE」 | **OFF** → ? |

**핵심 아하:** 스위치의 상태 글자(**ON / OFF**)를, 양쪽 괄호 사이에 **그대로 끼워 넣으면**
실제 영단어가 된다.

**풀이:**
- S + **ON** + G = **SONG** (스위치 ON)
- C + **OFF** + (double E = EE) = **C‑OFF‑EE = COFFEE** (스위치 OFF)
- (빈칸) + **OFF** + ICE = **OFF‑ICE = OFFICE** (스위치 OFF)

세 번째가 만드는 단어가 곧 답. (Notion 중급 해설의 정답 "OFFICE"와 일치.)

**정답: `OFFICE`**

### 중_7 — 트럼프 카드 + U/Z 방향 → **SEAL** ✅

**문제:** 2×2로 놓인 카드와 "영어로 답하세요 / U = 할인, Z = ????".
```
Spade 1 (♠A)   |   Heart 2 (♥2)
Diamond 3 (♦3) |   Club 2 (♣2)
```

**핵심 아하 (2단):**
1. 각 카드 = **무늬 영어 이름의 n번째 글자**.
   - Spade **1** → S**P**ADE 의 1번째 → **S**
   - Heart **2** → H**E**ART 의 2번째 → **E**
   - Diamond **3** → DI**A**MOND 의 3번째 → **A**
   - Club **2** → C**L**UB 의 2번째 → **L**
   - 네 글자 = S, E, A, L (위치는 좌상 S / 우상 E / 좌하 A / 우하 L)
2. **U · Z 는 카드를 읽는 "동선"** 이다.
   - **U자** 경로: 좌상→좌하→우하→우상 = S‑A‑L‑E = **SALE(할인)** ← 문제에서 준 힌트
   - **Z자** 경로: 좌상→우상→(대각선)좌하→우하 = S‑E‑A‑L = **SEAL**

"U = 할인(SALE)"이 방향 읽는 법을 가르쳐 주고, 같은 네 글자를 Z 동선으로 읽으면 답.

**정답: `SEAL`**

### 중_11 — 회색 숫자 = 로마 숫자 삽입 → **VIVA** ✅ (user 확인)

**문제:** 진한 글자와 흐린(회색) 숫자가 섞인 세 줄.
```
F 4 E = 5
S 9 = 6
6 5 A = ????   (답 4글자)
```

**핵심 아하:** 회색 숫자를 **로마 숫자로 바꿔** 진한 글자 사이에 그대로 끼워 넣으면
영단어가 된다. 오른쪽 `= 숫자`가 그 단어의 값이라 규칙을 확정해 준다.

- F **[4=IV]** E → F‑IV‑E = **FIVE** = 5 ✅
- S **[9=IX]** → S‑IX = **SIX** = 6 ✅
- **[6=VI] [5=V]** A → VI‑V‑A = **VIVA** (4글자) ← 정답

앞 두 줄(=5, =6)이 "회색 숫자 = 로마 숫자, 진한 글자 사이에 삽입"이라는 규칙을 가르치는
예시이고, 세 번째 줄이 실제 정답이다.

**정답: `VIVA`** *(예전 이 문서의 애너그램 추정 'SAFE'는 오답 — user가 로마숫자 규칙으로 정정)*

## 고수(상급)

### 상_1 — 🟡 (제공된 해설에 없음 · 직접 추론)

**문제:** 서로 다르게 분할된 원 3개(¼ 사분면 / 가로 분할 / 세로 반분)와 아래 표.
```
        원1    원2    원3
WE       -      8      4
CHILD    8      -      8
 ?       6      1      2      (세 글자로 답)
```

**내 분석:** 이 문제도 두 해설 페이지에 없다.
- 각 원은 글자 글리프의 **특정 영역(사분면 / 위·아래 / 좌·우 반쪽)** 을 가리키고,
  숫자는 그 영역에 걸치는 획·요소의 수를 세는 "영역 카운팅" 계열로 보인다
  (`WE`·`CHILD`가 규칙을 가르치는 예시, `?` 행이 실제 정답).
- 다만 원의 정확한 분수와 글리프 획 배치를 이미지만으로 확정할 수 없어,
  `?` 행 (6, 1, 2)을 세 글자로 확정하지는 못했다. **여기만 미해결로 남긴다.**

**정답: (미확정, 세 글자)**

### 상_10 — 조각 조합 + 크로스워드 → **ADOPT** ✅

**문제:** 방마다 단서(로마자 I·III·IV, "D하나", "5번의 답", "DIRTY의 반대"),
가운데 크로스워드 틀, 그리고 배터리 모양들과 숫자 칸(3·5 / 2·□·1 / 4·□).

**핵심 아하 (여러 단):**
1. 위쪽 조각 3개(가·나·다) 중 **2개를 겹쳐** 가운데의 배터리 모양 3개를 만든다 → 각 모양마다 소문제 1개가 드러난다.
2. 각 소문제는 "**단어들을 크로스워드에 채우고 빨간 화살표대로 읽어라**".
   - 단서 해독: "D하나" → **D**, "DIRTY의 반대" → **CLEAN**, "5번의 답"·"DONE/PEN" 등.
   - 숫자는 **단어 속 알파벳 위치**: D‑O‑N‑E의 1 = **D**, PE‑N의 3 = **N**, CLE‑A‑N의 4 = **A** → **DNA**
   - 나머지 두 조합 → **OT**, **PC**
3. 소문제 답 3개 = **OT / DNA / PC**. 문제에 표시된 **번호 순서(1~5)** 로 글자를 이으면:

```
OT / DNA / PC  →  번호대로 배열  →  A · D · O · P · T
```

**정답: `ADOPT`** (적응하다)

### 상_14 — 액자 속 알파벳 개수 → **TON** ✅

**문제:**
```
ONE   → ③ , W, H, F, U
TWO   → ① , R
THREE → ② , E
FOUR  → ✗
```

**핵심 아하:** 액자(그림 틀) 안에 있는 알파벳들의 **등장 횟수**를 세는 문제.
- **1번** 나오는 글자: N, W, H, F, U
- **2번** 나오는 글자: T, R
- **3번** 나오는 글자: O
- **4번(FOUR)** 나오는 글자: 없음 → ✗

각 줄의 동그라미 숫자 ①②③은 **정답에서의 자리(위치)** 다.
- ① (TWO 줄, 2번 등장군) → **T**
- ② (THREE 줄, 3번 등장군) → **O**
- ③ (ONE 줄, 1번 등장군) → **N**

W·H·F·U·R·E는 같은 횟수를 가진 "동점 글자"(교란용)이고, 단어가 되도록 T·O·N을 고른다.

**정답: `TON`**

### 상_17 — 가위바위보 · 손가락 개수 (답 2개) → **GROVE, SNAP** ✅

**문제:** 왼쪽에 보자기 손 "VS", 회색 상자 안에 세 손모양(주먹·가위·보) 열과 별표 칸.
`ANSWER = G★★★E , ★★★P` (콤마로 구분, 순서대로).

**핵심 아하 — 손이 두 가지 의미를 가진다:**

**① 회색 상자 안 단서만 → 손가락 = 숫자**
- 주먹 = 0 → **ZERO**, 가위 = 2 → **TWO**, 보 = 5 → **FIVE**
- 별(★)이 있는 자리의 글자를 뽑으면: ZE**R**O, TW**O**, FI**V**E → **R, O, V**
- G★★★E 틀에 넣으면 → **GROVE**

**② 문제 전체 단서 → 손 = 가위바위보 승부 (왼쪽 보자기와 대결한 결과)**
- 보 vs 주먹 → 보가 이김 = **WIN**, 보 vs 가위 → 보가 짐 = **LOSE**, 보 vs 보 → **DRAW**
- 별 자리 글자: LO**S**E, WI**N**, DRA**W**... → **S, N, A** (순서 맞춰)
- ★★★P 틀에 넣으면 → **SNAP**

**정답: `GROVE`, `SNAP`**

### 상_18 — 셀로판 색 겹치기 + 거꾸로 읽기 → **신발 (SMALL SHOES)** ✅

**문제:** 왼쪽에 색색의 LCD 글자 5줄(무지개색), 오른쪽에 뒤집힌 한글 메시지,
"숨겨진 암호는 작은 ??들".

**핵심 아하 (여러 단):**
1. 오른쪽 메시지는 **뒤에서부터 거꾸로** 읽는다 → "왼쪽에 셀로판 종이 다섯 장이 있다. **빠진 두 가지 색을 만들라**".
2. 배경이 무지개색인데, 셀로판(빨·노·파 등)에서 **빠진 두 색은 주황·초록**.
3. 색을 **겹쳐** 만든다(감산 혼합):
   - **빨강 + 노랑 = 주황** → 두 LCD 줄을 겹치면 글자 **SMALL** 등장
   - **파랑 + 노랑 = 초록** → 두 줄을 겹치면 글자 **SHOES** 등장
4. 숨은 암호 = **SMALL SHOES = 작은 신발들**.

**정답: `신발`** (숨은 문구 `SMALL SHOES`)
