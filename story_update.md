# STORY UPDATE - Project 03:14

## Core Concept

이 게임은 단순한 방탈출 게임이 아니다.

플레이어는 크래프톤 건물에서 깨어난다. 건물은 잠겨 있고, 시간은 계속
03:14를 가리킨다.

플레이어는 자신을 가둔 범인이 있다고 믿으며 건물에서 탈출하는 것을
목표로 행동한다.

그러나 실제 목표는 탈출이 아니다.

플레이어가 스스로 **03:14에 일어난 사건**을 기억해내는 것이다.

------------------------------------------------------------------------

# Theme

-   Psychological Horror
-   Mystery
-   Environmental Storytelling
-   Memory Reconstruction

공포는 귀신이나 괴물에서 나오지 않는다.

공포는 '기억이 잘못되어 있다는 사실'에서 나온다.

------------------------------------------------------------------------

# Player Belief

게임 시작 시 플레이어는 다음을 믿는다.

-   누군가 자신을 가뒀다.
-   범인은 계속 옥상으로 유도하고 있다.
-   건물 안에 범인이 남긴 흔적이 있다.

게임은 이 믿음을 계속 강화한다.

------------------------------------------------------------------------

# Actual Truth

플레이어를 가둔 사람은 존재하지 않는다.

옥상으로 올라간 것도 누군가에게 끌려간 것이 아니다.

플레이어는 자신의 의지로 옥상으로 향했다.

그 이유는 게임 내에서 직접 설명하지 않는다.

플레이어는 모든 증거를 종합하여 스스로 깨달아야 한다.

------------------------------------------------------------------------

# The Building

건물은 방탈출을 위해 만들어진 공간이 아니다.

건물 전체는 플레이어 자신이 만든 거대한 기억 저장소(Memory Palace)이다.

플레이어는 기억을 잃는 것을 알고 있었고, 미래의 자신이 반드시 진실을 알
수 있도록 건물 곳곳에 기억을 남겨두었다.

각 방은 하나의 퍼즐이 아니라 하나의 기억 저장 장치이다.

------------------------------------------------------------------------

# Meaning of Objects

## Notion

업무 관리 도구가 아니다.

미래의 자신에게 남긴 기억.

------------------------------------------------------------------------

## Git Commit

개발 기록이 아니다.

과거의 자신이 남긴 편지.

------------------------------------------------------------------------

## CCTV

감시 장비가 아니다.

사건의 증인.

------------------------------------------------------------------------

## Whiteboard

회의 내용이 아니다.

기억의 조각.

------------------------------------------------------------------------

## Meeting Notes

회의록이 아니다.

사건 당시의 사고 흐름.

------------------------------------------------------------------------

# Story Flow

## Act 1 - Escape

플레이어는 탈출을 시도한다.

건물에는 사람이 없다.

하지만 누군가가 있었던 흔적은 남아 있다.

모든 단서는 옥상을 가리킨다.

플레이어는 범인이 자신을 유인한다고 믿는다.

------------------------------------------------------------------------

## Act 2 - Suspicion

증거가 쌓인다.

그러나 모든 CCTV에는 플레이어 혼자만 등장한다.

출입 기록도 플레이어뿐이다.

플레이어는 처음으로

"왜 나는 스스로 올라가고 있었지?"

라는 의문을 갖는다.

------------------------------------------------------------------------

## Act 3 - Reconstruction

Git Commit

회의록

Notion

영상

출입기록

모든 단서가 서로 연결되기 시작한다.

플레이어는

누군가가 기억을 지운 것이 아니라

자신의 기억이 계속 무너지고 있다는 사실을 깨닫는다.

------------------------------------------------------------------------

## Act 4 - Memory Palace

옥상에서 노트북을 발견한다.

메모에는 다음과 같이 적혀 있다.

"나는 계속 잊게 된다.

그래서 이번에는 건물 전체를 기억장치로 만들었다."

플레이어는 처음으로

건물이 방탈출 맵이 아니라

자신이 설계한 기억 복원 장치라는 사실을 이해한다.

------------------------------------------------------------------------

## Final

난간 앞.

플레이어는 모든 기억을 되찾는다.

게임은

왜 옥상으로 올라왔는지

직접 설명하지 않는다.

대신 지금까지 수집한 모든 증거를 통해

플레이어 스스로

그 이유를 완성하도록 한다.

------------------------------------------------------------------------

# Narrative Rules

-   모든 퍼즐은 새로운 사실을 알려준다.
-   모든 공포 연출은 나중에 반드시 의미가 회수된다.
-   의미 없는 점프 스케어는 사용하지 않는다.
-   귀신이나 괴물은 등장하지 않는다.
-   "자살", "투신" 등의 단어는 직접 사용하지 않는다.
-   결말은 플레이어가 스스로 해석하는 구조를 유지한다.

------------------------------------------------------------------------

# Design Goal

플레이어가 게임을 끝냈을 때

"탈출했다."

보다

"내가 방금 하나의 사건을 추리해냈다."

그리고

"건물 전체가 처음부터 하나의 퍼즐이었다."

라는 감정을 느끼는 것을 최종 목표로 한다.
