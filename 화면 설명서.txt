2. 이동 화면
화면 ID : MY_002_AA_B
- 배경 UI
	- 화면 목적 : 플레이어가 맵의 특정 이동스팟으로 이동함에 따라 보이는 시야 변경을 배경화면 변경으로 표현함.
	- 이동 스팟 : 맵에서 플레이어가 존재할 수 있는(서 있을 수 있는) 스팟. 이동 스팟은 한 방 내에 여러개 존재 할 수 있음.
	- 각 배경 화면은 MY_002_AA_B 형태의 id를 가짐. AA는 맵에서의 이동 스팟 번호를 나타냄. B는 그 이동 스팟에서 보는 방향을 나타냄.
	- background 폴더 안에 있는 AA_B.jpg 형태의 파일들은 MY_002_AA_B 의 배경 화면과 매핑됨.
	- 각 키보드 방향키별 역할
		- 좌방향키 : B를 1늘림(B가 4라면 1로 만듦)
		- 우방향키 : B를 1줄임(B가 1이라면 4로 만듦)
		- 상방향키 : AA와 B에 따라서 결정된 다음 AA의 이동스팟으로 이동하고 B는 그대로 둠. 단, 현재 AA_B로 다음 이동스팟이 지정되지 않는다면 상방향키를 눌러도 이동하지 않음.
		- 하방향키 : 기능 없음
	- 이동 스팟 이동 시 화면 전환 방법 : 0.1초간 화면을 그라데이션으로 암전시켰다가 0.1초간 다음 배경 화면을 그라데이션으로 띄움

- 현재 연결된 배경 id 및 이동 스팟 
MY_002_01 과 MY_002_02 / 상방향키를 통한 이동 방법 : MY_002_01_03 -> MY_002_02_03 / MY_002_02_01 -> MY_002_01_01(-> : 상방향키 입력)
